# Contributing

Thank you for considering to contribute to this repository, and we would love to accept your contributions, whether code, issues or just general feedback.

## Before you start

- If you do not have any code contributions to make, consider either:
  - Creating an issue if you encounter any bugs, have any features to request, etc.
  - Contacting us for any generic feedback!
- If you do have code contributions to make, consider making an issue for this before creating a PR for changes, so that the issue can be discussed and guidance can be provided where possible.

In order to work with this repository, you will need NodeJS (v22 and above) and PNPM to work with this repository.

While [turborepo](https://turborepo.dev) is not required to get started working with this repository, it is used in CI and for builds, and is a recommended tool

> I am considering migrating completely to Bun, but because of initial contributors for this repo, we will go with incremental adoption of Bun.
